136/*!***************************************************************************
137 * @brief The measurement results data structure.
138 * @details This structure contains all information obtained by a single
139 * distance measurement on the device:
140 * - The measurement status can be read from the #Status.
141 * - A timing information is given via the #TimeStamp.
142 * - Information about the frame state is in the #Frame structure.
143 * - The 1D distance results are gathered under #Bin.
144 * - The 3D distance results for each pixel is at #Pixels or #Pixel.
145 * - Auxiliary values such as temperature can be found at #Auxiliary.
146 * - Raw data from the device is stored in the #Data array.
147 * .
148 *
149 * The pixel x-y orientation is sketched in the following graph. Note that
150 * the laser source would be on the right side beyond the reference pixel.
151 * See also \link argus_map ADC Channel Mapping\endlink
152 * @code
153 * // Pixel Field: Pixel[x][y]
154 * //
155 * // 0 -----------> x
156 * // | O O O O O O O O
157 * // | O O O O O O O O
158 * // | O O O O O O O O O (ref. Px)
159 * // y O O O O O O O O
160 * @endcode
