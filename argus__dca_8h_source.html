41 /*!***************************************************************************
42  * @defgroup argusdca Dynamic Configuration Adaption
43  * @ingroup argusapi
44  *
45  * @brief Dynamic Configuration Adaption (DCA) parameter definitions and API functions.
46  *
47  * @details The DCA contains an algorithms that detect ambient conditions
48  * and adopt the device configuration to the changing parameters
49  * dynamically while operating the sensor. This is achieved by
50  * rating the currently received signal quality and changing the
51  * device configuration accordingly to the gathered information
52  * from the current measurement frame results before the next
53  * integration cycle starts.
54  *
55  * The DCA consists of the following features:
56  * - Static or Dynamic mode. The first is utilizing the nominal
57  * values while the latter is dynamically adopting between min.
58  * and max. value and starting from the nominal values.
59  * - Analog Integration Depth Adaption (from multiple patterns down to single pulses)
60  * - Optical Output Power Adaption
61  * - Pixel Input Gain Adaption (w/ ambient light rejection)
62  * - ADC Sensitivity (i.e. ADC Range) Adaption
63  * - Power Saving Ratio (to decrease the average output power and thus the current consumption)
64  * - All that features are heeding the Laser Safety limits.
65  * .
66  *
67  * @addtogroup argusdca
68  * @{
69  *****************************************************************************/
