Detailed Description

Noise Detectors parameter definitions and API functions.

The module contains algorithms for noise detection and is comprised of 2 sections: Saturation Detector and Optical Disturber Detector. The Saturation Detector is designed to mark as saturated the pixels that have a too high phase noise (UCNoise). The ODD is designed to exclude pixels with bad SNR values or with phase noise (UCNoise) from the final result.
