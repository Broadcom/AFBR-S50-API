Detailed Description

SCI: Command definitions for log messages in printf style.

This file provides an implementation for log messages/commands in the printf style of the C standard library that can be used along with the systems communication interface.

This file provides command definitions for the systems communication interface.

static status_t vprint ( const char *  fmt_s,
va_list  ap 
)
inlinestatic

A vprintf() function for the uart connection.

This function is similar to print except that, instead of taking a variable number of arguments directly, it takes an argument list pointer ap. Requires a call to va_start(ap, fmt_s); before and va_end(ap); after this function.

Parameters
fmt_sThe printf() format string.
apThe argument list.
Returns
Returns the status (STATUS_OK on success).
